# homebrew-tap

Homebrew formulae that allows installation of Onlooker tools through the
[Homebrew](https://brew.sh) package manager.

## Installation

```sh
brew tap onlooker-community/tap
brew install <FORMULA>
```

## Formulae

| Repository                                                 | Formula                          | Description                |
| ---------------------------------------------------------- | -------------------------------- | -------------------------- |
| [onlooker](https://github.com/onlooker-community/onlooker) | [formula](./Formula/onlooker.rb) | CLI agent for Onlooker App |
